Callback-Funktion für eingehende MQTT-Nachrichten.

Diese Funktion wird aufgerufen, wenn eine neue MQTT-Nachricht empfangen wird. Die Sensorwerte aus der Nachricht werden in einem entsprechenden Objekt abgespeichert und in der Datenbank abgelegt, solange die Station dort vorhanden ist.
