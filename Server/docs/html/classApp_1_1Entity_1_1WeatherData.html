Detailed Description

Basisklasse für Sensordaten einer Wetterstation.

Diese abstrakte Klasse definiert die gemeinsamen Eigenschaften und Methoden für verschiedene Arten von Wetterdaten, die von einer Wetterstation erfasst werden können.
